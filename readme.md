# Awareness Chatbot

This project is a console chatbot that raises awareness about important topics.

## Features
- Chatbot conversation
- ASCII art display
- Voice greeting using WAV audio
- Console user interface

## Technologies
- C#
- .NET
- NAudio library

## How it works
The chatbot interacts with the user through the console and responds to user input.

The program also plays a voice greeting using a WAV file
